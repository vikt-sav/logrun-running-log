## ✨ Возможности

- **Несколько пользователей** — каждый со своей базой
- Автоматический расчёт **скорости** и **темпа**
- Сохранение истории в CSV-файлы
- Просмотр истории с возможностью удаления записей
- **Графики прогресса** (скорость, длительность, темп)
- Красивый и простой интерфейс
- Работает как обычная программа (`.exe` для Windows)

## 🛠 Как использовать

1. Запусти `Logran.exe`
2. Введи своё имя (или выбери существующего пользователя)
3. Добавляй пробежки:
   - Дата
   - Длительность (в формате `ММ:СС` или просто минуты)
   - Расстояние (км)
   - Место (опционально)
4. Смотри историю и графики прогресса
